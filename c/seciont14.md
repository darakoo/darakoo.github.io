01. C언어의 메모리 구조와 역할
	1) 전역변수, 지역변수
	[코드예시]
	int outer = 10;	// 전역변수
	void main(){
		int inner = 3;	// 지역변수
	}

	2) 메모리 구조
	Code Area : 프로그램의 코드
	Data Area : 전역변수, static변수 => 프로그램 종료시 해제
	Stack Area : 지역변수, 함수에 전달되는 인자 => 블럭{} 종료시 해제
	Heap Area : 개발자가 직접 메모리를 관리하는 영역(동적 메모리 할당 영역) => 개발자가 직접 해제
