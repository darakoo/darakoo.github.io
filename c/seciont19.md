01. 구조체
	1. 정의
	하나 이상의 변수를 묶어서 만든 새로운 유형의 사용자 정의 ???이다.
	반복되는 공통 요소를 쉽게 관리하기 위해서 주로 사용된다.
	
	2. 문법
		1) 기본문법
		struct 구조체이름{
			자료형 변수명,
			...
		};
		
		struct PERSON{
			char name[30];
			int age;
		};
		
		2) 구조체별명
		typedef struct {
			char name[30];
			int age;
		} PERSON;

	3. 선언 시 주의사항
		함수 밖에서 선언해야 한다.
		
02. 구조체와 배열
	- 구조체를 배열로 정의할 수 있다.
	struct PERSON boy[3];

03. typedef(type define) 선언
	1. 정의
	- 기존에 존재하는 자료형에 새 이름을 부여하는 것이다.
	- 복잡한 유형의 자료형 선언을 간결하게 처리 할 수 있다.
	
	2. 문법
	// 일반적인 변수선언에 적용시
	typedef int INT
	INT a = 0;	// int a = 0; 와 동일하다.
	
	// 구조체 적용시
	typedef struct 구조체이름{
		자료형 변수명,
		...
	}구조체별명;
